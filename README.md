# GitHub Introduction for Non-Developers

A friendly, multi-language GitHub Pages site designed to introduce GitHub to non-developers and IT professionals.

## Features

- **Beginner-Friendly Content**: Clear explanations of GitHub concepts without technical jargon
- **Multi-Language Support**: Available in English, Spanish, and French
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Key Topics Covered**:
  - What is GitHub?
  - Key concepts (Repositories, Commits, Branches, Pull Requests, Issues)
  - Getting started guide
  - Use cases for non-developers

## Viewing the Site

Visit the live site at: [https://maxjr2.github.io](https://maxjr2.github.io)

## Languages Supported

- English (en)
- Spanish (es)
- French (fr)

The site automatically detects your browser language and displays content accordingly. You can also manually switch languages using the language selector in the header.

## Local Development

To run this site locally:

1. Clone the repository
2. Open `index.html` in your web browser

No build process or special tools required!
